This is a simple test suite for testing common GUI elements of VPX.

Requirered:
- AutoIT 3 https://www.autoitscript.com/site/autoit/downloads/
- AutoIT Editor (optional) https://www.autoitscript.com/site/autoit-script-editor/downloads/
- AutoIT Recorder (optional). The recorder isn't part of the installer anymore. Download version 3.3.14.0. The recorder is located in the extra folder. https://www.autoitscript.com/autoit3/files/archive/autoit/autoit-v3.3.12.0.zip

Testrun:
- Start VPX and maximize the editor window.
- Load the simple example table (Ctrl+n)
- Under folder "tests" open one of the test file (e.g. Walltest.au3) in the AutoIT Editor or run it by right-click on the file and select "Run Script".
